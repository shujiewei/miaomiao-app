# 🐱 喵喵小百科

给爱猫的 9 岁小朋友的小程序。两种玩法：

| 模式 | 说明 |
|---|---|
| 🖥️ **浏览器试玩** | 双击 `index.html`，零安装零等待，开发改完立刻看 |
| 📱 **Android APK** | 通过 GitHub Actions 云构建，5–8 分钟出包，可安装到平板/手机 |

> **v0.2 更新**：增加领养系统！15 个家猫品种，答题攒小鱼干 🐟 去猫舍领养，每只猫都能自己起名字。

> **📱 想要 APK？** 看 [HOW_TO_BUILD_APK.md](./HOW_TO_BUILD_APK.md) 详细教程（零本地工具，全程浏览器操作）。

## 🚀 怎么打开（30 秒搞定）

### 方式 A：直接双击（最简单）
1. 用文件资源管理器打开 `C:\Users\shujiewe\miaomiao-app\`
2. **双击 `index.html`** — 用 Chrome / Edge 打开
3. 等 1 秒看到🐱 logo，就进入了

> 第一次打开需要联网下载 React + Tailwind CDN（约 5 秒），之后就可以离线用。

### 方式 B：手机上玩（更接近最终 APK 的体验）
1. 把整个 `miaomiao-app` 文件夹复制到任何手机里能访问的位置（OneDrive、微信文件传输助手都行）
2. 用手机浏览器（Chrome）打开 `index.html`
3. 在浏览器菜单里选 **"添加到主屏幕"**，就有了像 App 一样的图标

### 方式 C：开发模式（如果想边改边看）
```powershell
cd C:\Users\shujiewe\miaomiao-app
python -m http.server 8000
# 然后浏览器打开 http://localhost:8000
```

## 📁 项目结构

```
miaomiao-app/
├── index.html                      # 浏览器模式入口（用 CDN）
├── src/
│   ├── data.js                     # 内容数据
│   └── app.jsx                     # 主应用
├── package.json                    # Capacitor 依赖
├── capacitor.config.json           # APK 应用配置（包名/应用名）
├── scripts/
│   └── build-www.js                # 把源文件 + CDN 整合到 www/
├── resources/
│   └── icon.png                    # APK 图标（1024×1024 橘猫脸）
├── .github/workflows/
│   └── build-apk.yml               # GitHub Actions 云构建配置
├── HOW_TO_BUILD_APK.md             # ⭐ APK 构建完整教程
└── README.md
```

**构建产物（自动生成，不要手改也别上传）：**
- `www/` - 构建脚本生成的离线打包资源
- `android/` - Capacitor 生成的 Android 原生项目
- `node_modules/` - npm 装的依赖

这些都在 `.gitignore` 里，CI 每次重新生成。

**存储**：所有进度、作品、领养的猫都保存在浏览器 `localStorage` 里（key: `miao.user.v2`），换浏览器/清缓存会丢。设置页有"重置所有进度"按钮。

**家长测试入口**：关于页面有"🐟 +30 小鱼干"按钮，让你不用真的答完题也能体验领养流程。

## 🛠️ 已知限制（原型版）

- 动物用 emoji 占位，正式版会换成统一画风的 SVG 插图
- 音效是 Web Audio 现场合成，正式版会用真实喵叫
- 不支持多用户（一个浏览器一份进度）
- 画板保存为 PNG，但暂时只能存在 localStorage 里，不能导出

这些都会在 Stage 2 (Flutter APK) 解决。
